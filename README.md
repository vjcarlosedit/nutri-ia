
# Nutri-IA Expert App Design

This is a code bundle for Nutri-IA Expert App Design. The original project is available at https://www.figma.com/design/LtcMlNZjrsPZjjVzsqKtdT/Nutri-IA-Expert-App-Design.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Building for Android

This project uses Capacitor and Ionic Appflow for building Android APKs. See `CONFIGURACION_APK.md` for details.
